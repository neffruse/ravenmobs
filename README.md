# RavenMobs — Plugin for Paper 1.21.1

Level system for hostile mobs with kill rewards via Vault Economy.

## 📋 Features

- **Mob Levels** — every hostile mob receives a random level on spawn
- **Stat Scaling** — HP and damage increase with level
- **Kill Rewards** — money via Vault for every killed mob
- **Level Display** — level, name and HP shown above hostile mobs
- **All configurable** — all parameters customizable in config.yml
